1.1 算法

Algorithm E (Euclid's algorithm)

算法E 欧几里得算法

Given two positive integers m and n, find their gretest common divisor, that is, the largest positive integer that evenly divides both m and n.

给定两个正整数 m 和 n，求它们的最大公因子，即能够同时整除 m 和 n 的最大正整数。

E1. [Find remainder.] Divide m by n and let r be the remainder. (We will have 0 ≤ r < n)

E1 [求余数] 以 n 除 m 并令 r 为所得余数（0 ≤ r < n）

E2. [Is it zero?] If r = 0, the algorithm terminates; n is the answer.

E2 [余数为零？] 若 r = 0,算法结束，n 即为答案

E3. [Reduce.] Set m <- n, n <- r, and go back to step E1.

E3 [减少] 置 m <- n, n <- r，并返回 步骤 E1

一个算法只不过是一组有穷的规则，这些规则给出求解特定类型问题的运算序列；但除此之外，一个算法还有五个重要特征：

1) Finiteness.

1) 有限性

2) Definiteness.

2) 确定性

3) Input.

3) 输入

4) Output.

4) 输出

5) Effectiveness.

5) 可行性
